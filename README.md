A secure and user-friendly Password Generator web application built using Node.js, designed to help users create strong and customizable passwords instantly. This app provides flexible options that allow users to generate passwords tailored to their security needs.

Users can choose password length and include combinations of uppercase letters, lowercase letters, numbers, and special characters. The generated passwords are highly secure and suitable for protecting accounts across various platforms.

The application features a clean, modern, and responsive interface, ensuring seamless usability across desktops, tablets, and mobile devices. Additional functionalities like one-click copy and password strength indication enhance both convenience and security awareness.

🚀 Features
- Generate strong and secure passwords instantly
- Customizable options:
- Password length
- Uppercase letters
- Lowercase letters
- Numbers
- Special characters
- Password strength indicator
- One-click copy to clipboard
- Clean and responsive UI
- Cross-device compatibility

🛠️ Technologies Used
- Node.js – Backend runtime
- HTML5 – Structure
- CSS3 – Styling & responsiveness
- JavaScript (ES6) – Functionality
